Running Protractor scenarios in parallel

Sun, Sep 20, 2015
testing
Protractor can run tests in parallel, but only one feature file per Selenium instance. If a single feature file has more than one scenario, they cannot be parallelised out of the box. One solution to run scenarios in parallel is to write each new scenario in a new .feature file. However this breaks the natural grouping that feature files provide, and besides, it may already be too late if your team has many multi-scenario features.
